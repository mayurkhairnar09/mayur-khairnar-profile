## ğŸ¨ Customization

### Update Personal Information

1. **Hero Section**: Edit `src/components/Hero/Hero.jsx`
2. **About Section**: Edit `src/components/About/About.jsx`
3. **Skills**: Update `SKILL_CATEGORIES` array in `src/components/Skills/Skills.jsx`
4. **Experience**: Modify `EXPERIENCES` array in `src/components/Experience/Experience.jsx`
5. **Education**: Update data in `src/components/Education/Education.jsx`
6. **Projects**: Edit `PROJECTS` array in `src/components/Projects/Projects.jsx`
7. **Contact**: Update contact information in `src/components/Contact/Contact.jsx`

### Change Colors & Styles

